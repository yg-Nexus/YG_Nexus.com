# YG Nexus Website

**YG Nexus** – Next-generation AI Automation Studio

This repository contains the official website of **YG Nexus**, a studio specializing in AI chatbots, workflow automation, and custom web solutions that help businesses save time, reduce manual work, and increase conversions.

## Pages

- `index.html` – Home page  
- `automation.html` – AI Automation showcase page  
- `style.css` – Styling for the website

## Features

- Fully responsive and mobile-friendly  
- Smooth navigation between pages  
- Professional layout highlighting AI automation expertise  
- Lightweight and fast-loading code

## How to View

1. Clone or download this repository.  
2. Open `index.html` in your browser.  
3. For live demo, enable GitHub Pages via Settings → Pages → main branch → /root.
